# MikoUI

Include `framework/MikoUI/include/MikoUI.hpp` and link the `MikoUI` CMake INTERFACE target.

```cpp
#include <MikoUI.hpp>

int WINAPI WinMain(HINSTANCE, HINSTANCE, LPSTR, int) {
  AureliaUI::MikoUI::AppConfig cfg;
  return AureliaUI::MikoUI::Run(cfg, yourBundleSource, "app.js");
}
```
