# FontManager

**Header**: `include/utils/FontManager/FontMgr.hpp`

Singleton that loads system fonts and provides text rendering and measurement utilities. All text-drawing components delegate to this class internally.

## Named Font Constants

```cpp
static constexpr const char* DEFAULT_FONT  = "Segoe UI";
static constexpr const char* MONOSPACE_FONT = "Consolas";
static constexpr const char* EMOJI_FONT    = "Segoe UI Emoji";
```

## Class Reference

```cpp
class FontManager {
public:
    static FontManager& getInstance();

    void initialize();

    // Typeface & font object access
    sk_sp<SkTypeface> getTypeface(const std::string& familyName,
                                  SkFontStyle style = {});
    SkFont            createFont(const std::string& familyName,
                                 float size,
                                 SkFontStyle style = {});

    // Text rendering
    float drawText(SkCanvas* canvas,
                   const std::string& text,
                   float x, float y,
                   float fontSize,
                   const SkPaint& paint,
                   const std::string& family = DEFAULT_FONT);

    // Text measurement
    float measureText(const std::string& text,
                      float fontSize,
                      SkRect* bounds = nullptr,
                      const std::string& family = DEFAULT_FONT);

    void getFontMetrics(float fontSize,
                        SkFontMetrics* metrics,
                        const std::string& family = DEFAULT_FONT);

    sk_sp<SkFontMgr> getFontMgr() const;
};
```

### `drawText(...)`

Renders `text` at `(x, y)` (baseline origin) using the given `paint`. Returns the advance width of the rendered string.

### `measureText(...)`

Returns the advance width of `text`. Optionally fills `bounds` with the tight bounding rectangle.

### `getFontMetrics(...)`

Fills `metrics` with ascent, descent, leading, cap height, etc. for the given font family and size.

## Example

```cpp
auto& fm = FontManager::getInstance();

// Draw text manually inside a FlexNode::drawSelf override
void MyNode::drawSelf(SkCanvas* canvas) {
    SkPaint paint;
    paint.setColor(SK_ColorWHITE);
    float w = fm.drawText(canvas, "Hello", frame.x() + 8, frame.y() + 20,
                          14.0f, paint);
}

// Measure before layout
SkRect bounds;
float advance = fm.measureText("Hello World", 14.0f, &bounds);
```
