## Platform Support

### Windows ✅
Recommended: **Visual Studio 2026**

Build using:
```cmd
build.cmd
```

---

### macOS / Linux ⚠️
Not yet available out of the box.
You must build the Skia render backend manually for cross-platform support.

Fetch Skia via:
```sh
git submodule update --init external/skia
```
