# SPHXPainter

**Header**: `include/SPHXPainter.hpp`

`SPHXPainter` is a fluent 2D drawing wrapper over `SkCanvas`. Every method returns `*this` so calls can be chained.

Construct it inside any `FlexNode::drawSelf` override:

```cpp
void MyNode::drawSelf(SkCanvas* canvas) override {
    SPHXPainter p(canvas);
    p.roundRect(frame, 8.0f, SPHXColor::Hex(0x2d2d3f))
     .roundRectBorder(frame, 8.0f, SPHXColor::Hex(0x555577), 1.0f);
}
```

## Raw Canvas Access

```cpp
SkCanvas* raw() const;
```

Returns the underlying `SkCanvas*` for operations not covered by the fluent API.

## Example — Custom node

```cpp
void MyCard::drawSelf(SkCanvas* canvas) override {
    SPHXPainter p(canvas);

    // Drop shadow
    p.shadow(frame, 8.0f, SPHXColor::black().withAlpha(0.3f), 6.0f, 0, 4.0f);

    // Card background
    p.roundRect(frame, 8.0f, SPHXColor::Hex(0x1e1e2e));

    // Accent stripe
    SkRect stripe = SkRect::MakeXYWH(frame.left(), frame.top(), 4.0f, frame.height());
    p.roundRectCorners(stripe, 8.0f, 0, 0, 8.0f, SPHXColor::Hex(0x4080ff));

    // Border
    p.roundRectBorder(frame, 8.0f, SPHXColor::Hex(0x333355), 1.0f);
}
```
