# DAUx — Direct Audio Engine

**Header**: `include/SPHXDirectAudioEngine.hpp`

`DAUx` is SphereEngine's low-latency audio I/O engine. It wraps WASAPI (or DirectSound) and provides device enumeration, stream lifecycle management, a master effects chain, and a built-in mixer.

## AudioBackend

```cpp
enum class AudioBackend {
    Default,       // Platform default (WASAPI on Windows)
    WASAPI,
    DirectSound,
};
```

## AudioDevice

```cpp
struct AudioDevice {

    unsigned int              inputChannels;
    unsigned int              outputChannels;
    std::vector<unsigned int> sampleRates;
    unsigned int              preferredSampleRate;
    bool                      isDefaultInput;
    bool                      isDefaultOutput;
};
```

## MixerChannel

```cpp
struct MixerChannel {
    std::string name;
    float       volume = 1.0f;   // 0.0 – 1.0
    float       pan    = 0.0f;   // -1.0 (L) to +1.0 (R)
    bool        muted;
    bool        solo;

    std::vector<std::shared_ptr<AudioEffect>> effects;

    // Pull-mode audio source callback
    // Signature: (float* buffer, unsigned int frames, unsigned int channels)
    std::function<void(float*, unsigned int, unsigned int)> source;
};
```

## AudioEffect (Base)

```cpp
class AudioEffect {
public:
    virtual void process(float* buffer,
                         unsigned int frames,
                         unsigned int channels,
                         unsigned int sampleRate) = 0;
    bool enabled = true;
};
```

## Example — Sine wave output

```cpp
DAUx audio;

const float kSampleRate = 48000.0f;
float phase = 0.0f;

bool ok = audio.openDefaultStream(2, 48000, 256,
    [&](float* buf, unsigned int frames, unsigned int ch) {
        for (unsigned int i = 0; i < frames; i++) {
            float sample = std::sin(phase) * 0.5f;
            phase += 2.0f * M_PI * 440.0f / kSampleRate;
            if (phase > 2.0f * M_PI) phase -= 2.0f * M_PI;
            for (unsigned int c = 0; c < ch; c++)
                buf[i * ch + c] = sample;
        }
    });

audio.startStream();
```

## Example — Mixer channel

```cpp
MixerChannel ch;
ch.name   = "Synth";
ch.volume = 0.8f;
ch.pan    = -0.2f;   // Slightly left
ch.source = [](float* buf, unsigned int frames, unsigned int chans) {
    // Fill buf with your audio signal
};

auto limiter = std::make_shared<LimiterEffect>();
ch.effects.push_back(limiter);

int id = audio.addChannel(ch);
audio.setChannelVolume(id, 0.9f);
```
