# SphereEngine

**SphereEngine** is a modern, Windows-first C++ UI and audio framework for building high-performance native desktop applications. It combines GPU-accelerated 2D rendering, a flex layout engine, a JavaScript runtime, and a full-featured audio engine into one cohesive SDK.

## Getting started

After cloning, initialize/sync the repo submodules:

```powershell
.\scripts\submodules-init-sync.ps1
```

If you want to **skip Skia** (large checkout):

```powershell
.\scripts\submodules-init-sync.ps1 -SkipSkia
```

## Architecture

```
Skia (D3D12 GPU) + Yoga (Flex Layout)
           ↓
  SphereKit.Foundation
  (layout · events · theming · color · singletons)
           ↓
  SphereKit.GraphicInterface
  (Win32Window · IWindow abstract)
           ↓
  SphereKit.GraphicComponent
  (all built-in UI components)
           ↓
  SphereKit.JavaScriptEngine  (V8)
           ↓
  Sphere.React / SphereKit.Vue
```

## Key Technologies

| Technology | Role |
|---|---|
| **Skia** (D3D12 backend) | GPU-accelerated 2D rendering |
| **Yoga** (Facebook) | Flex layout engine |
| **V8** (Google) | JavaScript runtime for React/Vue UI |
| **WASAPI / DirectSound** | Audio I/O |
| **Win32 / DWM** | Window management & Windows 11 visual effects |

## Two UI Authoring Paths

### Native C++ — Imperative

Build the UI tree directly using `FlexNode` and component classes:

```cpp
#include <SPHXGraphicComponents.hpp>
#include <SPHXApplication.hpp>

int main() {
    Application::getInstance().init();

    auto window = SphereUI::createWindow();
    window->setTitle("My App");
    window->setSize(800, 600);

    auto root = FlexNode::Column();
    root->style.setPadding(16.0f);
    root->style.setGap(8.0f);

    auto label = std::make_shared<TextNode>();
    label->text = "Hello, SphereEngine!";
    label->fontSize = 20.0f;
    root->addChild(label);

    auto btn = std::make_shared<ButtonNode>();
    btn->label = "Click me";
    btn->onClick = [] { /* handle click */ };
    root->addChild(btn);

    window->setRoot(root);
    window->run();
    return 0;
}
```

### React / Vue — Declarative

Write components in TSX or Vue SFC, bundle them, then run inside `JavaScriptEngine`:

```cpp
#include <SPHXJavaScriptEngine.hpp>
#include <SphereReact.hpp>
#include "ReactUIResources.hpp"   // generated by gen_resources.py

int main() {
    Application::getInstance().init();

    JavaScriptEngine engine;
    engine.init();

    auto root = SphereUI::RenderReactApp(engine,
        std::string(ReactUIResources::bundle,
                    ReactUIResources::bundle + ReactUIResources::bundle_size));

    auto window = SphereUI::createWindow();
    window->setTitle("React App");
    window->setRoot(root);
    window->run();
    return 0;
}
```

## Module Headers

| Header | Contents |
|---|---|
| `SPHXFoundation.hpp` | Core foundation layer |
| `SPHXApplication.hpp` | Application singleton, ThemeManager |
| `SPHXGraphicInterface.hpp` | IWindow, Win32Window |
| `SPHXGraphicComponents.hpp` | All UI component headers |
| `SPHXColor.hpp` | Color API |
| `SPHXPainter.hpp` | Fluent 2D drawing API |
| `SPHXImage.hpp` | Image loading and rendering |
| `SPHXSvg.hpp` | SVG rendering |
| `SPHXJavaScriptEngine.hpp` | V8 embedder |
| `SphereReact.hpp` | React bridge |
| `SphereVUE.hpp` | Vue bridge |
| `SPHXDirectAudioEngine.hpp` | Audio I/O, MIDI, FFT |
| `SPHXJson.hpp` | JSON utilities |

## Build System

- **C++ Standard**: C++20
- **Compiler**: MSVC (Visual Studio 2022+), Ninja generator
- **Runtime**: Multi-threaded static CRT (matches Skia / V8)
- **SIMD**: AVX2
- **Build script**: `build.ps1`
- **Dependencies**: Skia + V8 prebuilts are auto-downloaded from GitHub releases

```powershell
./build.ps1
```
